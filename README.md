# E-Commerce Microservices

Struktur awal proyek microservice e-commerce menggunakan:
- Node.js (Express)
- PostgreSQL
- Docker

## Services

- api-gateway
- user-service
- product-service
- cart-service
- order-service
- payment-service
- notification-service
- inventory-service

## Cara Menjalankan

```bash
docker-compose up --build
```
